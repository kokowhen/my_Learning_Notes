# 创建数字列表

- 1.for 循环遍历列表时使用单复数相结合的形式有助于理解

- 2.创建数字列表numbers = list(range(a,b,c))#c是步长；当需要创建有一定规律的数字列表时，就可以用range直接生成

- **列表解析**：[New_n for n in list(range(a,b))]可以直接在列表内完成运算操作和赋值；可以简化代码

# 字典的遍历

- 键的遍历

  > ```
  > for i in dic.keys():
  > ```

- 值的遍历

  > ```
  > for i in dic.values():
  > ```

- 键值对遍历

  > ```
  > for keys,values in dic.items():
  > ```
